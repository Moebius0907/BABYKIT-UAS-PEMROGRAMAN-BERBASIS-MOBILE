# BabyKit - Aplikasi Pemesanan Produk Kebutuhan Bayi

Project ini merupakan tugas UAS mata kuliah **Pemrograman Berbasis Mobile**.  
Aplikasi dikembangkan menggunakan **Flutter** dan sudah terintegrasi dengan database menggunakan **PHP & MySQL**.

---

## Fitur Aplikasi

- Login dan Register
- Keranjang Belanja
- Tambah/Hapus Produk Favorit
- Riwayat Pesanan
- Detail Produk dan Checkout
- Pengelolaan Akun Pengguna

---

## Teknologi yang Digunakan

| Bagian | Teknologi |
|--------|-----------|
| Frontend | Flutter (Dart) |
| Backend | PHP (REST API) |
| Database | MySQL |
| Platform | Android |
